## Installation

To install BProgress, run the following command:

<Tabs groupId="packageManager" items={['npm', 'pnpm', "yarn", "bun"]} persist>
  <Tab value="npm">
    ```bash
    npm install @bprogress/remix
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @bprogress/remix
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @bprogress/remix
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @bprogress/remix
    ```
  </Tab>
</Tabs>

Now you are ready to use BProgress in your Remix application!

<Callout type="warn" title="Warning">
  This package requires Remix version 2.
</Callout>
